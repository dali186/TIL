## #1. `Azure Blob Storage`
---
#### 1. BlobStorage 란
> 구조화되지 않은 대량의 데잍터를 저장하는 데 최적화 (`비정형 데이터`, `이진 데이터`: 특정 데이터 모델이나 정의를 따르지 않는 데이터)

- 브라우저에 이미지 또는 문서 직접 제공
- 분산 액세스용 파일 저장.
- 비디오 및 오디오 스트리밍.
- 로그 파일에 쓰기
- 백업/복원, 재해 복구 및 보관용 데이터 저장
- 온-프레미스 또는 Azure 호스팅 서비스에서 분석하기 위한 데이터 저장.
