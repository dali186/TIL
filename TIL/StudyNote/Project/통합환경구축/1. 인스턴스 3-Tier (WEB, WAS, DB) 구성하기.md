#### 3. WEB/WAS/DB 설정
---
##### 1. WEB 설정
```
apt-get install apache2
systemctl start apache2
# 방화벽 설정
ufw app list
ufw allow 'Apache'
ufw allow 80/tcp
ufw enable
```
인바운드 포트를 Network Security Group에서 열어도 적용이 안됨 -> VM.네트워크 설정에서 적용
